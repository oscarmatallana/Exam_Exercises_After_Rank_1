# Exam Exercises After Rank 1

This repository contains my personal solutions and notes while preparing for a C programming exam.

The goals of this repository are to:

* Practice C programming fundamentals.
* Improve problem-solving and algorithmic thinking.
* Experiment with different implementations.
* Document my learning progress over time.

## Disclaimer

These solutions are shared **for educational purposes only** and reflect my personal learning process. They are **not intended to be copied, submitted, or used as official exam or project solutions**.

If you are preparing for a C programming exam, I encourage you to:

* Read and understand the exercise requirements.
* Develop your own solution before looking at other implementations.
* Use this repository to compare approaches, review concepts, or explore alternative ways of solving a problem.

The greatest learning comes from solving the exercises yourself.
