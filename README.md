# Exam Exercises After Rank 1

This repository contains my personal solutions and notes while preparing for a C Exam.

The goal of this repository is to:

* Practice C programming fundamentals.
* Improve problem-solving and algorithmic thinking.
* Experiment with different implementations.
* Track my learning progress over time.

## Disclaimer

These solutions are shared **for educational purposes only** and reflect my own learning process. They are **not intended to be copied or submitted as exam or project solutions**.

If you are preparing for a C Exam, I encourage you to:

* Read and understand the exercise requirements.
* Develop your own solution before looking at other implementations.
* Use this repository only to compare approaches, review concepts, or learn alternative ways of solving a problem.

Learning comes from solving the problems yourself.
